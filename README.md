### Fundamentos de JavaScript

#### O que é JavaScript?

JavaScript é a linguagem que deixa as páginas **interativas**.  
Na TechLoja usamos para:

- Mostrar produtos dinamicamente
- Controlar o carrinho
- Buscar e filtrar
- Comunicar com o PHP/MySQL

#### Variáveis

| Tipo     | Quando usar                  | Exemplo                     |
|----------|------------------------------|-----------------------------|
| `let`    | Quando o valor pode mudar    | `let nome = "Maria";`       |
| `const`  | Quando o valor **não** muda  | `const PI = 3.14;`          |

### Exercícios:

**1 – HTML/CSS (Período 1)**

1. Abra o arquivo `pratica_revisao.html`
2. Adicione uma seção chamada “Destaques da Semana”
3. Crie 3 cards de produto usando Grid
4. Defina pelo menos 2 variáveis CSS (`--primary` e `--radius`)

**2 – JavaScript (Período 2)**
No Console do navegador, crie:

1. Um objeto `meuProduto` com `id`, `nome`, `preco` e `categoria`
2. Um array `listaDeProdutos` com 3 objetos diferentes
3. Use template literal para mostrar no console:
   > “O produto [nome] da categoria [categoria] custa R$ [preco]”
